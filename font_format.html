CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation of HTML or XML documents. It defines how elements should be displayed on screens, paper, or other media, enabling developers to separate content (HTML) from design. This separation improves maintainability, reusability, and consistency across web pages.
